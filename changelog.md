# MAC OS SDK 版本说明

##  1.2版

该版本发布于2020-11-03。  

1、云端录制
2、动态修改分辨率
3、自定义视频采集
4、自动断线重连


##  1.1版

该版本发布于2019-7-24。  

1、增加桌面分享功能，并支持指定区域分享    
2、增加sdk 模式设置，区分测试 正式模式方便测试    
3、优化重连效果    

## 1.0 版

该版本发布于2019-7-12。  

1、支持1对1 1对多音视频通话 单房间支持最多15人 6人同时连麦    
2、支持纯音频通话模式    
3、抗丢包 15%    
4、断线制动重连    
5、支持配置自动发布模式
